# BotFather Commands Setup

Paste this into @BotFather via `/setcommands`:

```
start - Welcome screen and quick actions
link - Link your wallet address
status - Check your vault status
stats - View network statistics
notifications - Toggle alert notifications
email - Set or remove email for alerts
unlink - Unlink your wallet
help - How WillChain works
```

## Setup Steps

1. Open Telegram, go to [@BotFather](https://t.me/BotFather)
2. Send `/setcommands`
3. Select your WillChain bot
4. Paste the commands block above (without the ``` markers)
5. Done — users will see the command menu when they type `/`

## Optional: Description and About

```
/setdescription
WillChain — Digital inheritance on Base. Link your wallet to receive activity alerts and protect your vault.

/setabouttext
WillChain monitors your wallet activity and sends alerts when your vault approaches grace period. Link your wallet with /start to begin.
```
